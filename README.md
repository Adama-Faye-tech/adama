# Keylogger Éducatif

Ce projet est une plateforme d'apprentissage pour comprendre comment fonctionnent les enregistreurs de frappe (keyloggers) et, plus important encore, comment s'en protéger.

## 📁 Structure du Projet

- **mobile-android/** : Une application Android illustrant deux méthodes (Clavier personnalisé et Service d'accessibilité).
- **desktop-python/** : Un script Python simple pour Windows utilisant `pyHook`.
- **scripts/** : Scripts utilitaires pour Linux/Android (nécessite root).
- **docs/** : Documents pédagogiques sur le fonctionnement et la prévention.
